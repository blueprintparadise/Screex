# Screex v0.2 — Text-Aware Segmentation + Transcript Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make `screex index` segment screen recordings by on-screen *text change* (so subtle local UI changes no longer collapse to one state), and add a `screex transcript` command that emits a markdown step list.

**Architecture:** Add `segment_by_text` (text-keyed, with a motion pre-filter that only skips near-identical frames) alongside the existing motion `segment_stream`. `index` uses text mode by default with a `--fast` motion-only fallback. A new pure `transcript.py` formats a `ScreenIndex` to markdown, exposed via a `transcript` subcommand.

**Tech Stack:** Python, opencv-python, numpy, rapidocr-onnxruntime, pytest. Branch `improve/text-aware-segmentation` off latest `master` (includes the merged hardening work).

def segment_by_text(frames, text_fn, text_threshold: float = 0.80, motion_epsilon: float = 0.003):
    """Yield one Segment per text-distinct UI state.

    A cheap motion pre-filter skips OCR on frames that are essentially identical to the
    previous one (``motion < motion_epsilon``). Any frame that changed is OCR'd via
    ``text_fn(bgr) -> list[str]``; a new state begins when its text diverges from the
    current state's (``text_similarity < text_threshold``). The representative keyframe is
    the last (settled) frame of the state. Memory-bounded."""
    import cv2

    from screex.core.ocr import text_similarity

    prev_gray = None
    seg_idx = 0
    cur_start_t = None
    state_text = None
    last_t = None
    last_bgr = None

    for _idx, t, bgr in frames:
        gray = cv2.cvtColor(bgr, cv2.COLOR_BGR2GRAY)
        if prev_gray is None:
            cur_start_t = t
            state_text = text_fn(bgr)
        elif motion_score(prev_gray, gray) >= motion_epsilon:
            new_text = text_fn(bgr)
            if text_similarity(state_text, new_text) < text_threshold:
                yield Segment(seg_idx, cur_start_t, last_t, last_bgr, state_text)
                seg_idx += 1
                cur_start_t = t
            state_text = new_text
        prev_gray = gray
        last_t = t
        last_bgr = bgr

    if last_bgr is not None:
        yield Segment(seg_idx, cur_start_t, last_t, last_bgr, state_text)
```

from __future__ import annotations


def _fmt_time(seconds) -> str:
    s = int(round(seconds))
    return f"{s // 60}:{s % 60:02d}"


def format_transcript(screen_index) -> str:
    """Render a ScreenIndex to a markdown step transcript."""
    lines = [f"# Transcript — {screen_index.video}  ({_fmt_time(screen_index.duration)})", ""]
    for n, st in enumerate(screen_index.states, start=1):
        lines.append(f"## {_fmt_time(st.t_start)}–{_fmt_time(st.t_end)}  ·  State {n}")
        if st.ocr_text:
            lines.append(" · ".join(st.ocr_text))
        if st.text_added:
            lines.append(f"**Appeared:** {', '.join(st.text_added)}")
        if st.text_removed:
            lines.append(f"**Gone:** {', '.join(st.text_removed)}")
        lines.append("")
    return "\n".join(lines).rstrip() + "\n"
```

def test_index_text_mode_catches_subtle_change(subtle_screencast_video, tmp_path):
    from screex.cli import index
    from screex.core.index import ScreenIndex

    # default = text mode -> the "Loading" -> "Ready" change becomes a 2nd state
    p_text = index(str(subtle_screencast_video), fps=4.0, out=str(tmp_path / "t"), quiet=True)

    p_fast = index(str(subtle_screencast_video), fps=4.0, fast=True,
                   out=str(tmp_path / "f"), quiet=True)
    assert len(ScreenIndex.load(p_fast).states) == 1

# Screex v0.3 — OCR Speed + Robustness Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make `screex index` ~3.85× faster by configuring OCR threads, tolerate a single failing frame, and emit a clear message when a recording has no on-screen text.

**Architecture:** `ocr.extract_text` gains a `threads` param that builds `RapidOCR(intra_op_num_threads=…)` (default 2) and wraps the per-frame inference in try/except; `cli.index` plumbs `ocr_threads` everywhere, adds `--ocr-threads`, and logs a no-text guidance message after building.

**Tech Stack:** Python, opencv, numpy, rapidocr-onnxruntime, pytest. **CI lints with ruff — run `ruff check screex tests` before every commit, not just pytest.** Branch `improve/ocr-speed` off latest `master`.

```python
def _get_engine(lang=None, threads=2):
    """Return a cached RapidOCR engine for (lang, threads). ``threads`` sets
    onnxruntime's intra-op thread count (2 is ~3.85x faster than the library default on
    typical CPUs); ``threads=0`` uses the library default. ``lang`` is accepted for forward
    compatibility (RapidOCR's default models are multilingual)."""
    key = f"{lang or 'default'}:{threads}"
    engine = _engines.get(key)
    if engine is None:
        from rapidocr_onnxruntime import RapidOCR
        if threads:
            try:
                engine = RapidOCR(intra_op_num_threads=int(threads))
            except TypeError:
                engine = RapidOCR()
        else:
            engine = RapidOCR()
        _engines[key] = engine
    return engine
```

Replace the start of `extract_text` (the signature and the engine call) — change:

```python
def extract_text(bgr, lang: str | None = None) -> list:
```

to:
```python
def extract_text(bgr, lang: str | None = None, threads: int = 2) -> list:
    """..."""
    engine = _get_engine(lang, threads)
    try:
        result, _ = engine(bgr)
    except Exception:
        return []
    if not result:
        return []
```
